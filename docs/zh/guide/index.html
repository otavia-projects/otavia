详细指南

本节为 otavia 运行时的每个核心组件提供详细指南。

Actor 系统

  • Actor 模型 - Actor 层次、生命周期、邮箱调度、Barrier 机制
  • 消息模型 - 消息类型、Envelope、类型安全、Event
  • Stack 执行模型 - Stack 状态机、Future/Promise 链、对象池化
  • Address 模型 - Address 路由、PhysicalAddress、RobinAddress、线程亲和

IO 系统

  • Channel Pipeline - Handler 链、inbound/outbound 传播、executionMask

  • Reactor 模型 - NIO 传输层、Selector 引擎、SPI 机制

运行时

  • Actor IoC - BeanManager、依赖注入、autowire
  • Serde 框架 - 与 Buffer 集成的序列化/反序列化
  • SLF4A - 异步日志框架
